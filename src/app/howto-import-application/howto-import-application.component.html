1. Navigate to System Applications ⇒ Studio. Once you click on Studio, the application studio will open in separate browser tab.
  2. Click on the Import from Source Control button in the popup dialog.
  3. Enter {{repositoryUrl}} as the URL to the source repository. Enter the URL to the repository of the application to import.
  4. Enter credentials for connecting to the associated Git repository.
  5. Progress will be indicated as it imports the application. Once it is finished, click the Select Application button. ⇒
  6. You should now see the imported application listed. ⇒
